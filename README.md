# E-commerce Logistics Analytics Project

Projeto completo de análise de dados voltado para e-commerce e logística, utilizando dados públicos do Olist para construção de dashboards, análises exploratórias, KPIs estratégicos e modelagem de dados.

---

# Objetivo

Desenvolver uma solução de Analytics capaz de transformar dados brutos em insights estratégicos para apoio à tomada de decisão em áreas como:

- Logística
- Operações
- Comercial
- Marketing
- Produtos
- Atendimento ao cliente
